# 🎬 Movie Recommendation System

A Flask-based Content-Based Movie Recommendation System that recommends similar movies using Cosine Similarity and the TMDB 5000 Movies Dataset.

---

## 🚀 Features

- 🎬 Content-Based Movie Recommendation
- 🔍 Movie Search with Auto Suggestions
- 🖼️ TMDB Movie Posters
- ⭐ IMDb Ratings
- 📅 Release Year
- 🎭 Movie Genres
- 🌙 Netflix-style Dark UI
- ✨ Smooth Hover Animation

---

## 🛠️ Tech Stack

- Python
- Flask
- Pandas
- Scikit-learn
- HTML5
- CSS3
- TMDB API

---

## 📂 Dataset

- TMDB 5000 Movies Dataset
- TMDB 5000 Credits Dataset

## ▶️ Installation

```bash
git clone <repository-url>
cd Movie-Recommendation-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
